Dobot Magician

Dobot Magician is a multifunctional desktop robotic arm for practical training education. It was released by Shenzhen Yuejiang Technology in 2016. A model of the Dobot Magician was created with a suction cup as its end-tool. It can be easily programmed and controlled in the scene “UnitySimulationScene”. It can be manipulated for two kinds of actions via our application. One is called ‘New Movement’ and the other one is ‘New Suction Toggle’.

  • New Movement: This button adds a new "Move Robot To'' commands at the current position which enables the setting of movement targets at the specific point in code. When pressing the button, the coordinate is updated.
  • New Suction Toggle: This button adds a new "Toggle Suction Cup" command at the current position, which enables the toggling of the suction cup at the specific point in code.

"Make&Run Code" button compiles the current Touch Input commands. Afterwards, it is shown in the Code Panel and executed on the simulated Dobot. Additionally, the waypoints are shown, these represent the path in which the Dobot has to take.
