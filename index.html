Project Group Overview

The VARobot (Virtual and Augmented Reality Assisted Robot programming) project group aims to develop a solution combining the advantages of AR and robot simulation to implement an environment which assists robot programming. The group was conducted by the Computer Science Masters class of Paderborn University from March/2020 to April/2021.
Since robot programming is a very complex task, requiring specialist skills in multiple areas, we aim to make the programming and therefore use of robots more accessible for people without a robot programming background. To reach this goal, VARobot uses multi-platform AR as a possibility to program different types of robots as well as simulate the programs on a virtual robot displayed in the real environment. This way, collisions can be detected on the virtual model and without danger of damaging a real robot (or its environment). Our research was based on an earlier approach that was also developed at Paderborn University and focuses on extending this approach to introduce more flexibility in target robots and offer a multi-platform AR approach. As we want to keep our research practically applicable and usable in realistic situations, the framework also connects to physical robots to directly deploy developed programs.

SOLUTION IDEAS

Challenges

Challenging & Time Consuming

It is difficult to anticipate the robot’s movements in real 3D space and testing it takes time and can cause damage.

Complex Development Environments

Robot programming is done in very complex development environments, making trained professionals necessary.

Development Limited to PC

Development is usually done on PC, without flexibility in the choice of device.

One Development Environment per Robot

Robot programs are only useful for one specific robot and for a new robot, usually a new development environment has to be used and learned.

Limited Input Methods

No choice on input modality.

Steep Learning Curve

Steep learning curve in robot programming environments.

Solutions

Robot Simulation

Robot Simulation in the real world prevents the need to test on a real robot.

Responsive Design

Responsive design helps to reduce complexity of environment, flattening the learning curve.

Multi-platform Interface

Multi-platform interface offers the possibility to develop via smartphone or hands-free with HMD.

Diverse Target Robots

Diverse target robots offer the possibility to develop for different robots in one framework.

Voice Commands

Voice Commands (as well as AR interfaces) offer new diverse possibilities to interact with the programming environment.

Session Management

Session management makes it possible to easily switch between different scenarios, save progress or access old states Continue learning from point.
