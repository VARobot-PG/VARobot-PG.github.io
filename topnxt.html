LEGO Mindstorms NXT

LEGO MINDSTORMS is an educational robotics kit, enabling students to discover Science, Technology, Engineering and Mathematics in an enjoyable, engaging, and practical way. The robot can be built with its multiple parts which can be assembled together and programmed for attaining some objectives.
The ways to execute a program on a NXT robot have been listed in three broad steps.

  • Build the robot: There are specific robot building instructions laid down on a user guide provided along with the robot system.
  • Program the robot: The robot can be programmed using the LEGO MINDSTORMS Education NXT Software. Many examples are included in the Robot Educator part of the software. Programs can be downloaded to the NXT brick with the wireless Bluetooth connection or the USB cable.
  • Run your program: Finally, the program is exerted on the robot and it can be verified if the robot performs as expected. If not, one can try to adjust the robot or the program and it can be re-run.

The main components of a NXT robot with a brief description can be listed as below.

  • NXT Brick: The computer-controlled LEGO brick acts as the main operator of the LEGO MINDSTORMS robot.
  • Touch Sensors: It enable the robot to respond to obstacles in the environment.
  • Sound Sensor: It enables the robot to respond to sound levels.
  • Light Sensor: It enables the robot to respond to variations in light level and color.
  • Ultrasonic Sensor: It enables the robot to measure distance to an object and to respond to movement.
  • Lamps & Converter Cables: Additional lamps can be fit with the robot and then those can be programmed as flashing lights. It can also be used to activate the Light Sensor.
  • Interactive Servo Motors: The motors ensure that robots move smoothly and precisely.
  • Rechargeable Battery: It provides power to the NXT so the robot can move and respond to the instructions provided.
