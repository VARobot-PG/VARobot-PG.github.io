Multimodal Interaction

Our framework offers a variety of new ways for the user to interact with the programming environment in order to successfully program a robot. That is by gestures, using the user interface or even by voice via predefined voice commands.
The main idea behind introducing voice commands is to simplify the interaction of the users with the app on the supported devices.
There are particular voice commands that enable the user to navigate through the app, namely a voice command for each button which can be clicked and which has a functionality in the application. To trigger a button / function, all you need to do is press the “Speech Command” button and spell the name of the button as depicted below.
